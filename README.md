# Python Project Template

## Package manager
- poetry

## Tester
- pytest

## Package
- numpy

## How to run

### Activate poetry
```
poetry shell
```

### Deactivate poetry
```
deactivate
```

### Run python script
```
poetry run python same_project\main.py
```

## How to test

### From poetry
```
poetry run pytest
```

### From vscode test
- Press F1 key
- Python: Configure Tests
- framework: pytest
- directory : tests
